# 工作流定制

如何自定义 ComfyUI 工作流以实现特定功能。

---

## 工作流简介

Pixelle-Video 基于 ComfyUI 架构，支持自定义工作流。

---

## 工作流类型

### TTS 工作流

位于 `workflows/selfhost/` 或 `workflows/runninghub/`

### 图像生成工作流

位于 `workflows/selfhost/` 或 `workflows/runninghub/`

---

## 自定义工作流

1. 在 ComfyUI 中设计你的工作流
2. 导出为 JSON 文件
3. 放置到 `workflows/` 目录
4. 在 Web 界面中选择使用
