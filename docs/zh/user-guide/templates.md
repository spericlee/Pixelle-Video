# 模板开发

如何创建自定义视频模板。

---

## 模板简介

视频模板使用 HTML 定义视频画面的布局和样式。

---

## 模板结构

模板位于 `templates/` 目录，按尺寸分组：

```
templates/
├── 1080x1920/  # 竖屏
├── 1920x1080/  # 横屏
└── 1080x1080/  # 方形
```

---

## 创建模板

1. 复制现有模板文件
2. 修改 HTML 和 CSS
3. 保存到对应尺寸目录
4. 在 Web 界面中选择使用

---

## 模板变量

模板支持以下变量：

- `{{ title }}` - 视频标题
- `{{ text }}` - 分镜文本
- `{{ image }}` - 分镜图片
