# 自定义视觉风格

学习如何调整图像生成参数以创建独特的视觉风格。

---

## 调整提示词前缀

提示词前缀控制整体视觉风格：

```
Minimalist black-and-white illustration, clean lines, simple style
```

---

## 调整图像尺寸

不同尺寸适用于不同场景：

- **1024x1024**: 方形，适合小红书
- **1080x1920**: 竖屏，适合抖音、快手
- **1920x1080**: 横屏，适合B站、YouTube

---

使用「预览风格」功能测试不同配置的效果。
